# KoliBri - React with React Hook Form sample

This is a template for a KoliBri client side rendering application.
It demonstrates how to use [React Hook Form](https://react-hook-form.com/) with KoliBri components via the KoliBri React Hook Form adapter.

## Get Started

### Prerequisites

- [Node.js](https://nodejs.org/en/)

### Installation

Install the dependencies using npm:

```bash
npm i # or pnpm i or yarn
```

### Usage

#### Start the development server

```bash
npm start
```

#### Build the application

```bash
npm run build
```

#### Lint the code

```bash
npm run lint
```

#### Format the code

```bash
npm run format
```

#### Check unused code and dependencies

```bash
npm run unused
```

#### Run the tests

```bash
npm test
```
